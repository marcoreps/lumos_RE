serial ports:
/dev/ttyS2 seems to be the controller board for Z-axis motion control, start button supervision and the MOPA interface

/dev/ttyS4 seems to be the GD32F407 in LQFP144 package. boot default baud rate seems to be 500000.

the UV laser sources RS232 TX pin is level shifted and arrives at GD32F407 pin 116 (PD2). the laser sources RX pin comes from pin 113 (PC12). this matches UART5 in the STM32 context

rs232 communications with the UV source: 9600 8n1
when starting preview tracing:
command: SPK0\r\n
response: 0\r\n
when stopping preview tracing:
command: SPK0\r\n
response: 0\r\n
when preparing an engraving job, before sending the gocde file or starting it:
command: RG\r\n
response: 1\r\n
when sending the gcode file:
command: SPK0\r\n
response: 0\r\n
when canceling the job after sending, before executing:
command: SPK0\r\n
response: 0\r\n




sending $H0 to the MCU endpoint homes the Z axis and stays at home
sending $H1 to the MCU endpoint homes the Z axis and returns to where it was before
sending $RST to the MCU resets it
